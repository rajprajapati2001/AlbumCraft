# AlbumCraft

A lightweight, browser-based tool to organize images, edit metadata, and export them into a professional PDF layout. Perfect for documentation, field reports, or photo albums.

## ✨ Features

* **Smart Upload:** Drag and drop images or load sample files.
* **Live Editing:** * Rotate and flip images instantly.
    * Custom date and time formatting (supports MMM, YYYY, etc.).
* **Organization:** * Advanced sorting (Name, Size, Type, Date).
    * Mobile-friendly drag-and-drop reordering via dedicated handles.
* **High-Quality Export:** * One-click **PDF Download** and **Print** functionality.
    * Adjustable export quality scale.
    * Automatically hides UI tools (trash/rotate icons) during export.

## 🚀 Getting Started

To run this project locally without any server-side setup:

1.  **Download** the AlbumCraft Project.
2.  **Place your images** in the `gallery` folder/directory (if using Local Image).
3.  **Setup Script** Update [image_script.js](image_script.js) with your `image names` and `dates` to link them to the project.
4.  **Open with a Local Server:** * If using VS Code, right-click `index.html` and select **"Open with Live Server"**.
    * *Note:* Standard browser security may block sample images if opened via `file://` protocol.

## 🛠️ Built With

* **Frontend:** HTML5, CSS3 (Flexbox/Grid), FontAwesome 6.
* **Logic:** Vanilla JavaScript.
* **PDF Engine:** [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/)

## 📱 Mobile Usage

This app is optimized for mobile devices:
* Use the **Grip Icon** (`⋮⋮`) to drag and reorder images.
* Standard buttons are touch-optimized for easy rotation and deletion.
